Understanding CSS Border Properties

border-style

Defines the style of the border. Controls the visual appearance of the border line.

Common values: solid, dashed, dotted, double, groove, ridge, inset, outset

border-width

Specifies the thickness of the border. Can be set in pixels or using predefined values.

Values: thin, medium, thick, or specific length values (px, em, rem)

border-color

Sets the color of the border. Can be any valid CSS color value.

Values: hex, rgb, rgba, hsl, hsla, or named colors

border-radius

Rounds the corners of an element's border. Creates rounded corners or circular elements.

Values: length values (px, em, rem) or percentage

border (Shorthand)

Combines width, style, and color in one declaration. More efficient and cleaner code.

Syntax: border: width style color;

Example: border: 3px solid blue;
